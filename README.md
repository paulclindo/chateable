
# Personal Chat 

Basic chat for pets using React, Styled Components and Firebase

### Prerequisites

Install

```
npm install
```

### Installing

For development env running

```
npm run dev
```
For production env

```
npm run build
```

## Running the tests

```
npm run test
```

## Built With

React, webpack, Graphql & Jest
